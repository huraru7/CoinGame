# CoinGame

授業制作のコインプッシャーゲームです。

## ゲーム概要

コインプッシャーをモチーフにしたゲームです。
プッシャー（押し出し台）がZ軸方向に往復運動し、フィールド上のコインを押し出します。
落下したコインを回収し、スペースキーで新たなコインを投入して遊びます。

## 操作方法

| キー | 動作 |
|---|---|
| `Space` | コインを投入 |
| `A` | 投入位置を左に移動 |
| `D` | 投入位置を右に移動 |

## 使用技術

- **Unity** — ゲームエンジン
- **C#** — スクリプト言語
- **Unity Input System** — キーボード入力管理
- **Unity Object Pool** — コインの生成・再利用管理（メモリ効率化）
- **TextMeshPro** — UI（コイン枚数表示）
- **Rigidbody / Physics** — コインの物理挙動・プッシャーとの衝突
